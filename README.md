#  🧪 Manual Testing Project 

## 📌 Project Overview
This project is focused on **Manual Testing Project – E-Commerce Web Application** similar to Flipkart.  
The main objective is to verify core functionalities and document test results in a structured format.



## 📂 Modules Tested
- 🔐 Login Functionality  
- 🔍 Product Search  
- 🛒 Cart Management  
- 💳 Checkout Process  



## 🧾 Project Artifacts
This repository contains the following test documents:

- ✔ Test Cases Document (Excel)
- ✔ Bug Report Document (Excel)
- ✔ Screenshots for test execution proof

---

## ⚙️ Tools Used
- Manual Testing
- Microsoft Excel
- Web Browser (Flipkart Application Testing)

---

## 🎯 Objective
To understand real-world QA process by:
- Writing test cases
- Executing test scenarios
- Recording actual vs expected results
- Documenting bugs and system behavior



## 📊 Outcome
- Most test scenarios were executed successfully
- System behavior matched expected results in majority of cases
- Gained hands-on experience in **STLC (Software Testing Life Cycle)**



## 🚀 Key Learning
- Writing structured test cases
- Understanding Severity vs Priority
- Identifying expected vs actual behavior
- Real-time application testing mindset



## 📌 Note
This project is created for **learning and QA practice purposes** using a live e-commerce application.
